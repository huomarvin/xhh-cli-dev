# `@xhh-cli-dev/package`

> Package 包装类

## Usage

```
const Package = require('@xhh-cli-dev/package');

// 包安装目标路径
const local = '';
// 包版本
const packageVersion = '';
// 包名
const packageName = '';
// 包存储路径
const storePath = '';

const pkg = new Package({
    local,
    packageVersion,
    packageName,
    storePath
});

```
